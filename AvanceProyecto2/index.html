Encabezado H1: Titulos Principales

Encabezado H2: Subtitulos

Encabezado H3: Secciones menores

Los encabezados ayudan a dar jerarquia al texto respecto a tamaños

Es importante porque ayuda a identificar el nivel de cada texto o seccion de las pags

En este caso se uso del h1 hasta el h3 para detectar cada uno

Uso de Parrafos (Etiqueta p)

La etiqueta p se utiiliza como su etiqueta lo dice pues para parrafos

Podemos alinear parrafos y darles estilo para manejar mejor textos grandotes

Separar ideas en parrafos distintos siempre es buena practica incluso sino se trata de programacion

Enlaces e Hipervinculos (Etiqueta a)

Los enlaces pueden conectar diferentes paginas con otras secciones, paginas web, enlaces a otros sitios etc.

Puedes visitar sitios externos como GOOGLE para buscar mas info

El atributo "href" indica el destino, mientras que "target='_blank'" abre el enlace en una pestaña nueva.

Listas Ordenadas y No Ordenadas

Las listas ayudan a enumerar elementos de manera organizada, exites las ol y ul

Dentro de cada lista, utilizamos la etiqueta "li" para definir cada lista individual

Este proyecto hace uso de listas para la barra de nav y las imagenes de arriba

Formato de Texto negritas o italicas

Para resaltar texto importante podemos usar la etiqueta bold (negrita).

También existe la etiqueta italic (italica o cursiva) para dar enfasis

Incluso podemos subrayar (u) palabras.

Imagenes

Una imagen dice mas que mil palabras, siempre son un recurso fundamental

Se pueden representar con la etiqueta "img", donde "src="" " es el archivo al que hace referencia

"alt" nos sirve para saber a que se referia la imagen en caso de un error
